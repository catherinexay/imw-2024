Index Page Process

Framing the Project

For my index, I wanted to use pixel characters of myself to express my pages such as “exercises”, “projects”, and “reading”. I wanted it to be fun and simple as well as to have the repetition of pixels throughout the site as seen in the visuals, the buttons, and the font itself. Also, I wanted the format to be similar to a book and for the container sizes to respond to the window size so that it is accessible in various sizes.

Building the Index

For my first step, I created my pixel characters on Figma to figure out the container sizes for my pixel grids and the colours. I made sure that the container sizes were relatively all same height and had the same amount of rows for consistency.

Then I looked at different fonts from Google Fonts and Adobe Font. And I explored different fonts using the search 'pixel' and 'video game'.

Using colorhunt.co, I found my colour scheme for the index.

Using a Figma board, I designed the framework for all pages and elements of my site. Click on this link to view in detail.

The hardest part was figuring out how to create a decorative border for my buttons. But then I learned about border images which splits up an image in 9 pieces and depending on the amount of rounds/cycles, you can get interesting patterns from your image. So I had to create a png with my pixel colours to create the visual of a square pattern.
